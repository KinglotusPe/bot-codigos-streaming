------------------------------------------------------------
📊 ESTRUCTURA DE RANGOS (SISTEMA DE ACCESO)
------------------------------------------------------------

El bot permite escalar una red de distribución de códigos de forma jerárquica:

👑 OWNER (MÁSTER):
   - Control total de la base de datos de accesos.
   - Generación de rangos para distribuidores.
   - Visión global de todos los códigos procesados.

💼 SELLER (MAYORISTA):
   - Distribuidor de licencias de administración.
   - Crea y gestiona su propia red de gestores de códigos.
   - Ideal para modelos de negocio B2B.

🛠️ ADMIN (GESTOR DE ACCESOS):
   - El "operador" del sistema.
   - Registra los correos que el bot debe vigilar.
   - Asigna y renueva los accesos a los clientes finales.

👤 USER (USUARIO FINAL):
   - El beneficiario del código.
   - Recibe alertas automáticas cada vez que se genera un código de acceso o link de recuperación.

------------------------------------------------------------
🚀 FUNCIONES PRINCIPALES: CÓDIGOS AL INSTANTE
------------------------------------------------------------

🔹 EXTRACCIÓN INTELIGENTE DE OTP:
El bot no solo lee correos, ENTIENDE los códigos. Extrae automáticamente números de 6 dígitos, códigos alfanuméricos y botones de "Acceso Directo" para enviarlos limpios al usuario.

🔹 MONITOREO DE SEGURIDAD:
Vigilancia constante de bandejas de entrada. Ideal para:
- Netflix (Códigos de hogar/viaje)
- HBO Max (Verificación de inicio)
- Disney+ / Star+
- Amazon Prime (Links de aprobación)

🔹 HISTORIAL DE ACCESOS:
Cada código enviado queda guardado en un log privado. El usuario puede consultar /historial para recuperar códigos anteriores o verificar actividad.

🔹 GESTIÓN DE VENCIMIENTOS:
El sistema de códigos no es eterno. El bot controla cuándo vence el acceso de un usuario y le notifica automáticamente para que renueve su suscripción al código.
